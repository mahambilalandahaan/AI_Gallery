# AI Image Gallery

## Overview
This project generates AI images using Stable Diffusion in multiple styles (Realistic, Cartoon, Anime, Abstract) and displays them in a responsive HTML gallery with dark/light mode and download buttons.

## Tools & Libraries
- Python 3.x
- diffusers
- torch
- PIL (Pillow)
- Google Colab (optional)

## Gallery
The `AI_Gallery/index.html` file contains all generated images, style navigation, and download buttons.

## How to Run
1. Open `index.html` in a browser to view the gallery.
2. Download images using the buttons below each image.
